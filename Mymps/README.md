# Mymps蚂蚁分类信息 介绍
mymps（蚂蚁分类信息/地方门户系统）是一款基于php mysql的建站系统。为在各种服务器上架设分类信息以及地方门户网站提供完美的解决方案。 mymps,整站生成静态，拥有世界一流的用户体验，卓越的访问速度和负载能力。
